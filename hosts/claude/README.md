# Claude Host Package

Use this package when your environment supports native skills and you want Claude-oriented installation notes without forking the core behavior.

## Install

1. Import or upload the full Prompt Master folder in your Claude-compatible host.
2. Keep `SKILL.md` and `references/` together.
3. Invoke the skill naturally or by the host's explicit skill command if available.

## Notes

- The behavior source is the root `SKILL.md`.
- This folder exists only to document the Claude-oriented installation path.
- Do not customize this wrapper instead of the root skill unless the change is truly host-specific.
